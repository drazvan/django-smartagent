======
Django-SmartAgent
======
Description
------

:Author:
    James Pacileo

:Version:
    0.1.0

:Description:
    django-smartagent is an attempt to provide Django developers with a lightning fast and complete browser detection utility.


Installation
------

To install the app add `django_smartagent` to `INSTALLED_APPS`

    INSTALLED_APPS = (
        ...
        'django_smartagent',
        ...
    )
